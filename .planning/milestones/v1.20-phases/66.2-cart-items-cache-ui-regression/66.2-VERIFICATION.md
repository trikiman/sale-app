## Bug Evidence (Pre-fix)

User screenshot captured 2026-05-12: after tapping "add to cart" the card shows a clock icon (⟲) instead of the ± stepper. User reported that tapping "add" a second time shows the stepper correctly.

Root cause documented in 66.2-CONTEXT.md domain section. Fix is surgical: refuse to wipe the optimistic `cartItemsById` state with the empty items array that the Phase 63 cache returns.
