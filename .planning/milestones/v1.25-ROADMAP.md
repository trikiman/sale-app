## v1.25 Operator Visibility + Test Coverage (ACTIVE — started 2026-05-13)

Close the "time-to-notice" gap from v1.24. v1.24 reduced time-to-recover from ~60 min to ≤10 min, but the operator only learned about the original 2026-05-13 outage via opening the MiniApp. v1.25 fixes that via Telegram admin alerts + ops escape hatches + the integration tests the v1.24 verifier flagged.

| Area | Issue | v1.25 Phase |
|---|---|---|
| Time-to-notice | Operator learned about 60-min outage only via user-facing MiniApp | Phase 80 (OBS-08/09/10) |
