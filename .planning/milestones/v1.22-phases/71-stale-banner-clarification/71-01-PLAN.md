# 71-01 — App.jsx banner copy + per-source age label

**Phase:** 71 Stale Banner Clarification
**Requirements:** UX-COPY-01
**Depends on:** nothing (frontend-only)

## Goal

Rescope the stale-data banner from generic "Данные устарели" to "Источники устарели" (sources-scoped), and surface the per-source age inline (`зелёные (30 мин.)`) so the user sees which source file is old and by how much. Keep the `Обновлено: HH:MM` header label untouched — it correctly means "merged payload time".

## Change

### `miniapp/src/App.jsx` — `staleColorLabels` useMemo

Find the existing block (around line 1620):

```jsx
  const staleColorLabels = useMemo(() => {
    if (!sourceFreshness) return []
    const labels = {
      green: 'зелёные',
      red: 'красные',
      yellow: 'жёлтые',
    }
    return Object.entries(sourceFreshness)
      .filter(([, info]) => info?.isStale)
      .map(([color]) => labels[color] || color)
  }, [sourceFreshness])
```

Replace with:

```jsx
  const staleColorLabels = useMemo(() => {
    if (!sourceFreshness) return []
    const labels = {
      green: 'зелёные',
      red: 'красные',
      yellow: 'жёлтые',
    }
    return Object.entries(sourceFreshness)
      .filter(([, info]) => info?.isStale)
      .map(([color, info]) => {
        const base = labels[color] || color
        // v1.22 UX-COPY-01: surface per-source age inline so the banner
        // is self-explanatory (e.g. "зелёные (30 мин.)") and no longer
        // contradicts the merged-payload `Обновлено` header.
        if (info?.status === 'missing') {
          return `${base} (нет файла)`
        }
        if (typeof info?.ageMinutes === 'number') {
          return `${base} (${Math.round(info.ageMinutes)} мин.)`
        }
        return base
      })
  }, [sourceFreshness])
```
