**Phase:** 69 Drift Visibility

**Depends on:** 69-01 (helpers must exist)

## Goal

Call `_compute_xray_drift_block` from `_build_reliability_snapshot` and integrate its reason + critical flag into the existing severity mapping. Also extend `pool_refresh_complete` event with `success_rate_drops: list[str]`.

## Change

### `backend/main.py::_build_reliability_snapshot`

Locate the cart_add block section (`cart_add_block, cart_add_reason = _compute_cart_add_block()`) and add BELOW it:

```python
    # v1.21 OBS-06: xray config drift visibility
    drift_block, drift_reason, drift_is_critical = _compute_xray_drift_block(
        pool, cycle_age
    )
    if drift_reason:
        reasons.append(drift_reason)
```

Then modify the critical-set expression. Current:

```python
    critical = {"xray_bridge_not_listening", "no_cycle_state"}
    has_critical = any(r in critical for r in reasons)
```

Change to:

```python
    critical = {"xray_bridge_not_listening", "no_cycle_state"}
    has_critical = any(r in critical for r in reasons) or drift_is_critical
```

Finally attach the block if present (below the existing `cart_add` attachment):

```python
    if drift_block is not None:
        snapshot["xray_drift"] = drift_block
```

### `vless/manager.py::refresh_proxy_list`

Before the `pool_refresh_complete` emit, capture pre/post dead-node state:

```python
        # OBS-07 completion: identify hosts that went from alive→dead in
        # this cycle (REL-15 grading). Compared against the BEFORE state
        # captured pre-rewrite so we only count transitions, not
        # already-dead nodes.
        success_rate_drops = sorted(
            n["host"]
            for n in (new_pool.get("nodes") or [])
            if n.get("host") and self._is_node_dead(n)
            and n["host"] not in {
                h for h in (old_dead_hosts := ...)
            }
        )
```

Actually simpler: capture `old_dead_hosts` BEFORE the pool rewrite, then compute diff after:

```python
        # Before the pool rewrite in Phase 68 code (after filtering, before
        # replace_nodes). Insert this BEFORE the running_before line:
        old_dead_hosts = {
            n["host"] for n in (self._pool.get("nodes") or [])
            if n.get("host") and self._is_node_dead(n)
        }

        # ... existing running_before + admitted_hosts_after + pool rewrite ...

        # After rewrite:
        new_dead_hosts = {
            n["host"] for n in (new_pool.get("nodes") or [])
            if n.get("host") and self._is_node_dead(n)
        }
        success_rate_drops = sorted(new_dead_hosts - old_dead_hosts)
```

Add to the `pool_refresh_complete` payload:

```python
                "success_rate_drops": success_rate_drops,
```

```python
def test_pool_refresh_complete_includes_success_rate_drops(pm, monkeypatch):
    """pool_refresh_complete carries the list of hosts newly demoted to dead."""
    from vless.parser import VlessNode
    new_fake = VlessNode(uuid="u", host="fresh-1", port=443, name="n")
    # Put an existing dead host in the old pool by seeding 20 failures.
    pm._pool["nodes"] = [{"host": "existing-alive", "port": 443}, {"host": "already-dead", "port": 443}]
    for _ in range(20):
        pm.record_outcome("already-dead", success=False)
    # Admission returns fresh-1 + re-admits already-dead; post-refresh,
    # already-dead is still dead (20 failures carry over), so no NEW drop.
    # Add a new-dead host that the admission returns:
    new_dead = VlessNode(uuid="u2", host="fresh-dead", port=443, name="n2")
    for _ in range(20):
        pm.record_outcome("fresh-dead", success=False)

    monkeypatch.setattr(pm, "_reload_xray_systemd", lambda: ("ok", 1, None))
    monkeypatch.setattr("vless.manager.sources.fetch_igareck_list", lambda: "x")
    monkeypatch.setattr("vless.manager.sources.parse_vless_list", lambda t: ([new_fake, new_dead], []))
    monkeypatch.setattr("vless.manager.sources.filter_ru_nodes", lambda ns: ([new_fake, new_dead], []))
    monkeypatch.setattr(pm, "_probe_candidates_in_parallel", lambda c: [new_fake, new_dead])
    monkeypatch.setattr(pm, "_rebuild_and_restart_xray", lambda: None)

    pm.refresh_proxy_list()

    lines = [json.loads(line) for line in pm._events_path.read_text().splitlines() if line.strip()]
    completes = [e for e in lines if e.get("event") == "pool_refresh_complete"]
    assert len(completes) == 1
    evt = completes[0]
    assert "success_rate_drops" in evt
    # fresh-dead is newly dead in the new pool (20 failures recorded before admission)
    assert "fresh-dead" in evt["success_rate_drops"]
    # already-dead was dead before and after, so not in drops
    assert "already-dead" not in evt["success_rate_drops"]
```
